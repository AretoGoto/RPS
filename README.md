# Игра Камень-Ножницы-Бумага

## Описание
Консольная игра "Камень-Ножницы-Бумага" на Python. Игрок играет против компьютера до 5 побед.
Реализован подсчёт очков и сохранение итогового результата в файл.

## Как играть
1. Откройте терминал в папке проекта
2. Запустите игру командой:
   ```bash
   python game.py
3. Сделайте выбор:

1 — камень

2 — ножницы

3 — бумага

4. Для выхода из игры нажмите 0

## Функциональность

Выбор хода игрока

Случайный выбор хода компьютера

Проверка корректности ввода

Возможность выхода в любой момент

Подсчёт побед игрока и компьютера

Сохранение результата игры в файл scores.txt

## Требования

Python 3.x
